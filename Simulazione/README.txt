First, install amqp.node using npm:

npm install amqplib

Eseguire il file JS send_data.js per avviare la simulazione dell'acquisizione dati da campo.

Il file JSON_EXAMPLE contiene alcuni casi d''uso, la struttura del json, se modificata, deve ssere modificata anche sulla macchina CEP.
In tale ottica , ogni modifica sulla struttura dei JSON in ingresso sulla macchina CEP comporta una modifica dell' INput Stream e dell'execution plan associato all'inputstream.
